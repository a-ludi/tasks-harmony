# F02: Local Data ZIP Backup Export and Import — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let the user export all local data as a `tasks-harmony-backup-YYYY-MM-DD.zip` file and import it back, replacing all current data.

**Architecture:** A new `src/backup/backup.ts` module wraps the existing `exportAppState` / `importAppState` functions with ZIP packaging via `fflate`. The ZIP contains a single `state.json` file. `ProfilePage.tsx` gains two buttons — Export and Import (file picker). Import validates the parsed JSON with the existing `validateAppState` schema before applying. The store's `reload()` is called after import to refresh all state.

**Tech Stack:** `fflate` (`zipSync`, `unzipSync`, `strToU8`, `strFromU8`), existing `sync/export.ts`, `sync/import.ts`, `schemas/validate.ts`, React `useRef` for the hidden file input.

- [ ] **Step 3: Implement backup.ts**

Create `src/backup/backup.ts`:

```ts
import { zipSync, unzipSync, strToU8, strFromU8 } from 'fflate';
import type { AppState } from '@/types';

const STATE_FILENAME = 'state.json';

export function buildBackupFilename(dateStr: string): string {
  return `tasks-harmony-backup-${dateStr}.zip`;
}

export function wrapStateInZip(state: AppState): Uint8Array {
  const json = JSON.stringify(state, null, 2);
  return zipSync({ [STATE_FILENAME]: strToU8(json) });
}

export function unwrapStateFromZip(zipBytes: Uint8Array): AppState {
  const files = unzipSync(zipBytes);
  const stateBytes = files[STATE_FILENAME];
  if (!stateBytes) throw new Error(`ZIP does not contain ${STATE_FILENAME}`);
  return JSON.parse(strFromU8(stateBytes)) as AppState;
}
```

Add the export handler inside the component:

```ts
async function handleExport() {
  if (!db) return;
  const state = await exportAppState(db);
  const zipBytes = wrapStateInZip(state);
  const date = new Date().toISOString().substring(0, 10);
  const filename = buildBackupFilename(date);
  const blob = new Blob([zipBytes.buffer as ArrayBuffer], { type: 'application/zip' });
  const url = URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = filename;
  a.click();
  setTimeout(() => URL.revokeObjectURL(url), 100);
}
```

Add the import handler:

```ts
async function handleImport(e: React.ChangeEvent<HTMLInputElement>) {
  const file = e.target.files?.[0];
  if (!file || !db) return;
  setImportError(null);
  setImportSuccess(false);

  try {
    const buffer = await file.arrayBuffer();
    const zipBytes = new Uint8Array(buffer);
    const state = unwrapStateFromZip(zipBytes);
    const validation = validateAppState(state);
    if (!validation.valid) {
      setImportError(`Invalid backup file: ${validation.errors.join('; ')}`);
      return;
    }
    await importAppState(db, state);
    await reload();
    setImportSuccess(true);
  } catch (err) {
    setImportError(err instanceof Error ? err.message : 'Failed to import backup');
  } finally {
    // Reset the file input so the same file can be re-selected
    if (fileInputRef.current) fileInputRef.current.value = '';
  }
}
```
